Intro

Note: This is not a WordPress theme, and it wont install like a WordPress theme, this is a HTML5 template (static website).

Clinicom is a Responsive quick and easy customizable Modern Multipurpose Medical & Health Template that comes with 18+ valid HTML5 pages and 8 Home page variations. Clinicom offers numerous awesome features like , clinic, cardiology, surgery, dentist, optometrist, laboratory, general hospital, cosmetic surgery, pediatric clinic, veterinary, psychiatrist, physiatrist or your just a Blog Our demos work for one doctor office and private practices with more medical specialtie. Well commented code, well-documented file structure and 24/7 professional and ultra-fast Support.

To edit this template with your info you will need a code editor, i use Sublime Text (http://www.sublimetext.com/3) and a browser i recommend to use Google Chrome.

It is really easy to use. There is multiple home page layouts to chose, also various selection of pages that you can use to create your website, but you can also create your own page layouts very easily (just copy and paste code form the elements examples).

When you start editing the template with your info is good idea to create separate folder, copy the assets folder and 404.html page there, rename one of the index... files to index.html and then move and edit pages that you will use. Also is good practice to create separate styleseet for your edits and load it bellow Clinicom CSS.
